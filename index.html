Programming Projects

Python: CAD .dwg to .gdb

This python code was developed to help bring layers from a CAD (.dwg) file to a geodatabase (gdb) in ArcGIS Pro. The client had 197 schools, and each school had its own folder with the CAD floorplans. After georeferencing, this program would automate bringing each floorplan into a feature dataset created for each school in a gdb in ArcGIS Pro

Main Processing

  1. Loop through each school folder.
  2. Check if feature dataset already exists in gdb, and if not, create feature dataset in gdb
  3. Find .dwg datasets in each school folder
  4. Extract Point and Polyline layers from each .dwg dataset
  5. Rename and import layers to feature dataset in gdb.
  6. Using a search cursor, extract specific features with ROOMTAG attribute and export as new feature class in the feature dataset
